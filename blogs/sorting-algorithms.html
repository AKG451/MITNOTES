Sorting Algorithms: QuickSort, MergeSort, and HeapSort

Sorting algorithms are fundamental to computer science and are used to arrange elements in a specific order. This comprehensive guide covers three of the most efficient and widely-used sorting algorithms: QuickSort, MergeSort, and HeapSort.

Each algorithm has its own advantages and trade-offs in terms of time complexity, space complexity, and practical performance characteristics.

Algorithm

QuickSort Algorithm

QuickSort is a divide-and-conquer algorithm that works by selecting a 'pivot' element and partitioning the array around it.

  • Choose a pivot element from the array
  • Partition the array so elements smaller than pivot are on the left, larger on the right
  • Recursively apply QuickSort to the sub-arrays
  • Time Complexity: Average O(n log n), Worst O(n²)
  • Space Complexity: O(log n)

MergeSort Algorithm

MergeSort is a stable divide-and-conquer algorithm that guarantees O(n log n) time complexity.

  • Divide the array into two halves
  • Recursively sort both halves
  • Merge the sorted halves back together
  • Time Complexity: O(n log n) in all cases
  • Space Complexity: O(n)

HeapSort Algorithm

HeapSort uses a binary heap data structure to sort elements efficiently.

  • Build a max heap from the input array
  • Repeatedly extract the maximum element
  • Place it at the end of the sorted portion
  • Time Complexity: O(n log n)
  • Space Complexity: O(1)

Program (Code)

QuickSort Implementation

def quicksort(arr, low, high): # Base case: if low >= high, array is sorted if low < high: # Partition the array and get pivot index pi = partition(arr, low, high) # Recursively sort elements before and after partition quicksort(arr, low, pi - 1) quicksort(arr, pi + 1, high) def partition(arr, low, high): # Choose the rightmost element as pivot pivot = arr[high] # Index of smaller element i = low - 1 for j in range(low, high): if arr[j] <= pivot: i += 1 arr[i], arr[j] = arr[j], arr[i] # Place pivot in correct position arr[i + 1], arr[high] = arr[high], arr[i + 1] return i + 1 # Example usage arr = [64, 34, 25, 12, 22, 11, 90] quicksort(arr, 0, len(arr) - 1) print("Sorted array:", arr)

MergeSort Implementation

def mergesort(arr): if len(arr) > 1: # Find the middle point mid = len(arr) // 2 # Divide the array into two halves left_half = arr[:mid] right_half = arr[mid:] # Recursively sort both halves mergesort(left_half) mergesort(right_half) # Merge the sorted halves i = j = k = 0 while i < len(left_half) and j < len(right_half): if left_half[i] < right_half[j]: arr[k] = left_half[i] i += 1 else: arr[k] = right_half[j] j += 1 k += 1 # Copy remaining elements while i < len(left_half): arr[k] = left_half[i] i += 1 k += 1 while j < len(right_half): arr[k] = right_half[j] j += 1 k += 1 # Example usage arr = [64, 34, 25, 12, 22, 11, 90] mergesort(arr) print("Sorted array:", arr)

HeapSort Implementation

def heapify(arr, n, i): # Find largest among root and children largest = i left = 2 * i + 1 right = 2 * i + 2 if left < n and arr[left] > arr[largest]: largest = left if right < n and arr[right] > arr[largest]: largest = right # If root is not largest, swap and continue heapifying if largest != i: arr[i], arr[largest] = arr[largest], arr[i] heapify(arr, n, largest) def heapsort(arr): n = len(arr) # Build max heap for i in range(n // 2 - 1, -1, -1): heapify(arr, n, i) # Extract elements one by one for i in range(n - 1, 0, -1): # Move current root to end arr[0], arr[i] = arr[i], arr[0] # Call heapify on reduced heap heapify(arr, i, 0) # Example usage arr = [64, 34, 25, 12, 22, 11, 90] heapsort(arr) print("Sorted array:", arr)
