Title

Binary Search Algorithm: Efficient Search in Sorted Arrays

Binary search is one of the most fundamental and efficient searching algorithms in computer science. It works by repeatedly dividing the search interval in half, making it significantly faster than linear search for sorted data.

This algorithm is essential for understanding divide-and-conquer techniques and forms the basis for many advanced algorithms and data structures.

Algorithm

Binary Search Algorithm Steps

Binary search works on the principle of divide and conquer. It requires the array to be sorted beforehand.

  • Start with the entire sorted array
  • Find the middle element of the current search space
  • Compare the target value with the middle element
  • If target equals middle element, return the index
  • If target is less than middle, search the left half
  • If target is greater than middle, search the right half
  • Repeat until element is found or search space is empty

Complexity Analysis

  • Time Complexity: O(log n) - logarithmic time
  • Space Complexity: O(1) for iterative, O(log n) for recursive
  • Best Case: O(1) - element found at middle
  • Worst Case: O(log n) - element at extremes or not present

Prerequisites

  • Array must be sorted in ascending or descending order
  • Elements should be comparable
  • Random access to elements (arrays, not linked lists)

Program (Code)

Iterative Binary Search Implementation

def binary_search(arr, target): # Initialize left and right pointers left = 0 right = len(arr) - 1 while left <= right: # Calculate middle index mid = (left + right) // 2 # Check if target is present at mid if arr[mid] == target: return mid # If target is greater, ignore left half elif arr[mid] < target: left = mid + 1 # If target is smaller, ignore right half else: right = mid - 1 # Target not found return -1 # Example usage arr = [2, 3, 4, 10, 40, 50, 80, 100] target = 40 result = binary_search(arr, target) if result != -1: print(f"Element found at index {result}") else: print("Element not found")

Recursive Binary Search Implementation

def binary_search_recursive(arr, left, right, target): # Base case: element not found if left > right: return -1 # Calculate middle index mid = (left + right) // 2 # Element found if arr[mid] == target: return mid # Search in left half elif arr[mid] > target: return binary_search_recursive(arr, left, mid - 1, target) # Search in right half else: return binary_search_recursive(arr, mid + 1, right, target) # Example usage arr = [2, 3, 4, 10, 40, 50, 80, 100] target = 40 result = binary_search_recursive(arr, 0, len(arr) - 1, target) if result != -1: print(f"Element found at index {result}") else: print("Element not found")

Binary Search Variations

# Find first occurrence of target def find_first_occurrence(arr, target): left, right = 0, len(arr) - 1 result = -1 while left <= right: mid = (left + right) // 2 if arr[mid] == target: result = mid right = mid - 1 # Continue searching left elif arr[mid] < target: left = mid + 1 else: right = mid - 1 return result # Find last occurrence of target def find_last_occurrence(arr, target): left, right = 0, len(arr) - 1 result = -1 while left <= right: mid = (left + right) // 2 if arr[mid] == target: result = mid left = mid + 1 # Continue searching right elif arr[mid] < target: left = mid + 1 else: right = mid - 1 return result # Example with duplicates arr = [1, 2, 2, 2, 3, 4, 5] target = 2 first = find_first_occurrence(arr, target) last = find_last_occurrence(arr, target) print(f"First occurrence at index: {first}") print(f"Last occurrence at index: {last}")
